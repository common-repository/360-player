Dies ist der Grundpfad, von dem aus nach den beiden Filelists gesucht wird.
Format: [Basepath]/[Object]/[Filelist path]
Beispiel: http://example.com/360_pictures/ cake / small/Filelist.txt
Für diese Option gibt es Shortcuts, die einen Teil des Pfades herleiten:
"__SITE__": Verweist auf die URL der Startseite.
"__PLUGIN__": Verweist auf den Haupt-Ordner dieses Plugins, wie er auch für die Ressourcen benutzt wird.
"__UPLOAD__": Verweist auf den Upload-Ordner, der in Wordpress verwendet wird.
