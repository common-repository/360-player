Benutzung

Dieses Plugin ist dazu gedacht den Thrixty-Player in Ihre Wordpress Seite einzubinden.
Um den Player einzubinden, können Sie beliebig viele Player und Layouts anlegen und diese später mit Hilfe unseres Einbindungstools in Ihre Website einfügen.

Wenn Sie einen Player einbinden möchten welcher sein eigenes Set an Optionen hat, können Sie hierfür einen Player anlegen.
Wenn Sie mehrere Player einbinden möchten, welche sich die selben Optionen teilen, können Sie hierfür ein Layout anlegen.

Dieses Plugin arbeitet mit Shortcodes, um diese in Ihre Seite einzubinden gibt es ein Einbindungstool, dies finden Sie in der Tool-Bar des Editors.

Player

Ein Player besteht aus den Pfadinformationen für die File-Listen und allen weiteren Optionen.
Durch das Verwenden von Playern können Sie einen Player in Ihre Seite einbinden dessen Einstellungen einzigartig sind.
Die erstellten Player können Sie hier verwalten, um einen neuen Player hinzuzufügen klicken Sie bitte hier.

Layouts

Ein Layout besteht nur aus den Optionen(Exklusive Pfadinformationen).
Wenn Sie das Layout in die Seite einbinden wollen, müssen Sie zusätzlich die Pfade für die File-Listen angeben (Dies passiert erst beim Einbinden des Shortcodes).
Durch das Verwenden von Layouts können Sie beliebig viele Player mit den gleichen Einstellungen in ihre Website einbinden.
Wenn Sie dann die Einstellungen des Layouts ändern, wirkt sich das auf alle Player aus, die dieses Layout verwenden.
Die erstellten Layouts können Sie hier verwalten, um ein neues Layout hinzuzufügen klicken Sie bitte hier.

Player und Layouts einbinden

Um ihre bereits erstellten Player und Layouts einzubinden, können Sie unser Einbindungstool verwenden.
Dieses Tool finden Sie in der Tool-Bar im Wordpress Editor.
Beim Einbinden können Sie wählen ob Sie ein Player oder ein Layout einbinden möchten.
