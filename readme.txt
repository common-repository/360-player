Mit diesem Plugin können sie Thrixty Player Instanzen in ihre Seite einbinden

== Description ==
Dieses Plugin ist dazu gedacht den Thrixty-Player in Ihre Wordpress Seite einzubinden.
Um den Player einzubinden, können Sie beliebig viele Player und Layouts anlegen und diese später mit Hilfe unseres Einbindungstools in Ihre Website einfügen.

== Installation ==
1. Plugin Herunterladen
2. Plugin Aktivieren
3. Fertig !!!
